Rc

Macro-based reference counting for standalone libraries.

Use Rc.define to generate concrete Rc/Weak types for one payload type. The generated types are single-threaded and follow Rust-like Rc/Weak drop semantics.

define

macro

Macro 

                        (define name value-type)

Generate managed Rc and Weak reference-counted types.

Example:

(Rc.define RcString String)

This creates:

  • type RcString
  • type RcStringWeak
  • module RcString with strong-reference operations
  • module RcStringWeak with weak-reference operations
  • automatic copy and delete implementations for both.
